# blink65
Write Arduino-style sketches for VIC-20 and Commodore 64.

## Blink - BASIC
This exapmple is written without the **blink65** library.

Although it has the same structure of an Arduino sketch, it shows how the I/O lines of the User Port can be programmed from BASIC.

This example is specific for Commodore 64, 
but porting it to other Commodore systems is as simple as changing the addresses of the Data Direction Register
and of the Port A or Port B Register.
