Arduino-style API

This example summarizes the Arduino-style API implemented so far:

void setup(void)
{
pinMode(LED_BUILTIN, OUTPUT);
}
void loop(void)
{
digitalWrite(LED_BUILTIN, HIGH);
delay(1000);
digitalWrite(LED_BUILTIN, LOW);
delay(1000);
}

_(1)_ Pins of the upper row differ significantly between systems. Some of them can be used as additional input or output lines.

_(2)_ Pins B and M of the lower row are available on all systems, but with different constraints.

I/O pins from C to L are connected to one port of a VIA or CIA chip on all systems and can be independently programmed as input or output.
